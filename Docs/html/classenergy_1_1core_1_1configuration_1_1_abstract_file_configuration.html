Detailed Description

Абстрактный класс конфигурации, частично реализующий интерфейс IConfiguration. Класс описывающий конфигурации хранящиеся в локальных файлах на диске. Реализует функционал открытия и закрытия файла. Файл читается целиком, поэтому не рекомендуется использовать для объемных файлов конфигураций.
