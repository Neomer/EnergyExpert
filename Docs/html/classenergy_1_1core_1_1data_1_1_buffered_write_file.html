Detailed Description

Класс для работы с файлами через буфер в оперативной памяти для операций записи в файл. Подходит для оптимизации скорости работы приложения для случаев, когда данные в файл пишутся часто и небольшими порциями, при этом актуальность данных в файле не важна. Например, класс идеально подходит для логирования.
