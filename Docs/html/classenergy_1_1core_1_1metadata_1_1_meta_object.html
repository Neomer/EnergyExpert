Detailed Description

Абстрактный класс для предоставления доступа к метаданным класса. Класс, предоставляющий метаданные, должен быть унаследован от класса MetaObject и реализовать виртуальный метод MetaObject::getMetadata(), возвращающий метаданные объекта.
