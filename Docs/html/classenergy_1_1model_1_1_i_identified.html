Detailed Description

template<typename T>
class energy::model::IIdentified< T >

IIdentified<T> интерфейс для всех классов, имеющих идентификатор. Тип T - базовый тип или класс, который переопределил оператор сравнения для корректной работы метода IIdentified::isEquals(const IIdentified<T> &other).
